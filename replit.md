# Lush Engineering Website

## Overview
A static website for Lush Engineering, an electrical and construction engineering company based in Francistown, Botswana.

## Project Structure
```
project/
├── index.html       # Homepage
├── about.html       # About page
├── services.html    # Services page
├── projects.html    # Projects gallery
├── contact.html     # Contact page
├── main.js          # Main JavaScript
├── QUICK-START.md   # Quick start guide
└── assets/
    ├── css/
    │   └── styles.css
    ├── img/
    │   └── (project images)
    └── js/
        └── main.js
```

## Running the Website
The website is served using Python's built-in HTTP server on port 5000.

## Features
- Pure HTML/CSS/JS (no backend required)
- Responsive design for mobile, tablet, and desktop
- WhatsApp integration for contact
- Project gallery with filters and lightbox
- SEO-ready with proper meta tags

## Deployment
Configured for static deployment on Replit.
